# LLM Get Started
Get started with LLMs.

First, Install 

* Python 3.11
* Pytorch 2.6+
* ollama by installer - [https://ollama.com/download](https://ollama.com/download).

Next, clone codes.

```
git clone https://github.com/hailiang-wang/llm-get-started.git
cd llm-get-started
```
